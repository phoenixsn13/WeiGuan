# 围观 P14 入口接线审计

日期：2026-07-06

本清单用于审核「发起 → 世界 → 身份 → 评论区 → 复盘 → 历史」是否能被用户逐步到达。可见文字只使用世界名、身份名、内容摘要和中文功能名；存储编号只保留在路由里，不作为按钮文字或标题展示。

## 入口矩阵

| 从 | 到 | 当前入口 | P14 后入口 | 验收方式 |
|---|---|---|---|---|
| 发起页 | 世界 | 新建世界或继续世界的选择区 | 填写可选世界名，或从搜索列表选择已有世界 | 新建提交带 `world_name`；继续提交带 `world_id` 且不覆盖名称 |
| 世界总览 | 世界全景 | 世界卡里的「世界全景」 | 不带发起筛选进入同一世界的全量现场 | 点击后进入 `/world/{world_id}/live` |
| 世界总览 | 身份页 | 世界卡主身份名 | 身份名按钮进入 `/identity/{person_id}?world_id={world_id}` | 锚文本是身份名，无裸编号 |
| 历史记录 | 身份页 | 历史分组作者名 | 作者名按钮进入 `/identity/{person_id}?world_id={world_id}` | 锚文本是作者名，无裸编号 |
| 评论区 | 身份页 | 有人物归属的作者名 | 作者名进入 `/identity/{person_id}?world_id={world_id}`；无人物归属不强造入口 | 有归属才出现身份入口，未知作者保持普通显示 |
| 多平台现场 | 身份页 | 有人物归属的作者名 | 作者名进入 `/identity/{person_id}?world_id={world_id}`；无人物归属不强造入口 | 从世界事件和人物账户能对应时才接线 |
| 现场 | 复盘 | 「看结果」 | 保持既有入口 | 完成后可进入复盘页 |
| 历史记录 | 评论区/复盘 | 历史条目操作按钮 | 保持评论区、回放、平台现场入口 | 点击按钮进入对应内容页面 |

## 当前实现状态

- 已接线：发起页世界选区、世界总览世界入口、世界总览主身份入口、历史记录作者入口、历史记录到评论区/复盘。
- 条件接线：评论区和多平台现场的作者入口只在能拿到人物归属时启用；无人物归属不强造入口。
- 可见文字检查：世界名与身份名均来自可显示名，不展示裸存储编号。
