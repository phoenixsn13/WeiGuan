# P8 Retro 原型自审

日期：2026-07-03

## 产物

- `frontend/prototypes/retro-analysis.png`
- 生成方式：内置 imagegen，单张合成图，包含桌面版与移动版。

## 自审结论

- 禁用词：画面未出现 `agent`、`OASIS`、`仿真`、`工作台`、`trace`、`simulation`、`dashboard`。
- 产品调性：以「围观回放 / 发酵时间线」表达复盘台，避免把用户直接推入技术或系统管理心智。
- 信息架构：左侧是发帖身份和原帖上下文，中间承载真实动力学，右侧承载创作者洞察卡；移动端保留同一信息层级。
- 方法族：原型覆盖传播树、立场分化、影响力榜、情绪时间线、数据趋势五类入口。
- 配色：深墨壳、暖琥珀传播线、绿/红/灰立场色，未使用图表库默认彩虹配色。
- 洞察文案：以「关键传播节点」「立场拐点」「发酵半衰期」「极化风险」「建议」表达，不直接展示公式。

## 实现计划摘记

- Retro 主区消费 `GET /api/runs/{id}/analysis`，禁止前端假造指标。
- 洞察卡由 deterministic projection 翻译成人话，保留现有 LLM insights 接口不改。
- 五个视图使用同一世界层视觉语言：暖琥珀用于传播/连接，绿/红/灰仅用于立场。
- 空态必须诚实：没有级联、没有拐点、没有结构节点时直接说明，不补固定波次。
